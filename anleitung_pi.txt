Pi konfigurieren mit Pi OS Lite (64-bit, ohne Desktop) – Pi 4 empfohlen.

User: taf


=== Dateien auf den Pi kopieren ===

scp mediaplayer_app.py taf@<IP>:
scp config_autostart.sh taf@<IP>:


=== WLAN konfigurieren ===

sudo -i
wpa_passphrase "TaF-Kasse" "EIA-TIA-568B-Cm" >> /etc/wpa_supplicant/wpa_supplicant.conf
exit


=== Installation und Autostart einrichten ===

sudo bash config_autostart.sh


=== Testen ===

sudo systemctl start taf_service.service
systemctl status taf_service.service
journalctl -u taf_service.service -f    # Live-Log


=== Verzeichnisstruktur auf dem USB-Stick ===

Saisonen und Produktionen werden als Unterverzeichnisse erwartet:

  USB-Stick/
  ├── 2023-24/
  │   ├── Faust/
  │   │   ├── bild1.jpg
  │   │   └── bild2.jpg
  │   └── Hamlet/
  │       └── ...
  ├── 2024-25/
  │   └── ...
  ├── medienbasis/      ← Bilder, die immer im Grundstock bleiben (flach)
  │   └── logo.jpg
  └── skripte/          ← Python-Dateien werden ins Home-Verzeichnis kopiert

USB-Stick anstecken → Daten werden kopiert → Stick kann abgezogen werden.
Der Player springt danach automatisch zur Spielzeiten-Übersicht.


=== Bedienung (Touch-Display) ===

1. Spielzeit antippen
2. Produktion antippen → Diashow startet
3. ‹ / › : manuell blättern
4. "← Zurück": eine Ebene zurück


=== Vorschaubilder ===

Das erste Bild im Produktionsordner wird automatisch als Vorschau verwendet.
Soll ein bestimmtes Bild als Vorschau dienen, einfach alphabetisch als erstes benennen (z.B. _preview.jpg).
Thumbnails werden unter ~/.thumbs gecacht und bei jedem USB-Update neu generiert.


=== Hinweise zur Hardware ===

Pi 4 / Pi 5:   volle Unterstützung, Touch, Diashow
Pi Zero / 2:   zu langsam für Kivy – bitte play_mediafiles.py (altes Skript) verwenden
